# PortfolioOS Future-Only Backlog

This backlog is locked after the PortfolioOS v1 maintenance freeze. It is not a
roadmap continuation and does not authorize implementation by itself.

## Future Real Historical SUE Panel

Status: locked. WRDS/PIT required.

Any real historical SUE panel must use point-in-time data, historical universe
membership, event availability timestamps, tradable timestamps, delisting
handling, and replayable data manifests. Deterministic expanded fixtures cannot
be reclassified as real historical evidence.

## Future Paper-Overlay Calibration

Status: locked. Not alpha promotion.

Paper-overlay work can only calibrate execution-environment behavior such as
latency, spread capture, drift, and reconciliation. It cannot promote SUE,
revision, a composite, or any future alpha family.

## Future Factor Discovery Import Review

Status: locked. Phase 64 only.

Future Factor Discovery artifacts can enter PortfolioOS only through the Phase
64 research import contract. Teaching-mode or sandbox artifacts cannot route
directly into Q1 evidence, Promotion Gate, Q2, paper-stage workflows, or Alpha
Registry state changes.

## Future Live/Broker/Order Work

Status: locked. Explicit approval only.

Any future work touching live connectivity, execution venues, account state, or
order-like payloads requires a separate human decision, a written boundary pack,
and focused validation before any code path is opened.

## Future Production Approval Path

Status: locked until real historical evidence and governance exist.

Production approval requires real historical evidence, independent review,
governance, replayable manifests, risk controls, operational controls, and a
separate approval process. The current v1 release does not grant that path.

## Backlog Rule

New work requires explicit reopen decision. No automatic roadmap expansion after
Phase 66.
