# Alpha Discovery Cycle Closeout One-Pager (2026-04-17)

## Cycle Framing

This cycle was run as a two-family alpha-discovery campaign from charter design through closeout, not as a frozen factor tournament.

The goal was to determine whether the discovery machine could honestly distinguish:

- a calibration family that should teach the machine how to judge signals,
- from a primary family that might produce a true winner.

The cycle finished with a full closeout and no promoted winner.
That outcome is still a successful discovery result because the cycle produced reusable machine assets, prevented false-positive promotion, and left the primary family in a bounded restart state rather than an ambiguous dead end.

## What Was Done

### 1. Calibration Family

Calibration was run on:

- `US residual momentum / residual reversal`

Its purpose was not to produce a final alpha package.
Its purpose was to test whether the discovery machine itself could tell signal from artifact under residualization and adversarial falsification.

That work produced two durable machine assets:

- `P-001`: exposure-conditioned adversarial nulls
- `FM-001`: null-consistent residualization strengthening

Together they hardened the discovery process against a class of false positives that would otherwise have looked like residualized alpha.

### 2. Primary Family

Primary mining was then run on:

- `A-share state-transition microstructure`

This family was chosen because the edge comes from market design and state transitions, not from a generic crowded anomaly.

The cycle produced:

- a mechanism charter
- an adversarial and ideal-data plan
- a provider-backed A-share state-transition daily-panel path
- a live upper-limit pilot chain with matched controls, pre-event placebo, and conditioned nulls

The first live family slice was fully read and formally closed out.

## Actual Research Output

This cycle produced four durable outputs.

1. A calibrated discovery machine
   - the null system is now exposure-aware rather than naively unconditional
   - residualized expressions can now be judged against a more honest adversarial baseline

2. A reusable machine-governance layer
   - `P-001` and `FM-001` are reusable principles, not family-specific notes
   - they apply to future families that change sample exposure through residualization, neutralization, or event conditioning

3. A real A-share research path
   - provider-backed daily panel construction now exists
   - state-transition pilot artifacts can be emitted on real historical data rather than synthetic fixtures only

4. A bounded family conclusion
   - the primary family did not drift into narrative inflation
   - it closed with an honest category:
     - `mechanism suggestive but unproven`

## Why It Did Not Advance

The primary family did not advance to winner status for three concrete reasons.

1. The strongest upper-limit expressions showed local positive separation, but did not beat their own pre-event placebo windows cleanly.
2. The downside branch (`M3`) was too sparse in the live window to support a serious family winner claim.
3. The regime gate could not be honestly cleared because the current provider path was not operationally viable for a wider pre-`R3` historical unlock inside this cycle.

This is why the final classification is not `winner`, but also not `nothing there`.

It is:

- `mechanism suggestive but unproven`

## What Carries Forward

This cycle should not be remembered as `tried and failed`.
It should be remembered as:

- a successful discovery cycle that improved the machine,
- prevented a likely false positive,
- and left the primary family in a restartable intermediate state.

If the family is reopened later, the restart prior is not blank.
The inherited starting points are:

- `P1/P4` directionality is worth retesting

The correct executive summary is:

- no winner was promoted,
- but the cycle materially improved the research machine and produced a valid, reusable closeout state.
